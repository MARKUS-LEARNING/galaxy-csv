# Data Sources

## music_library.csv

- **Row count:** ~64,000
- **Columns:** Song, Artist, Album, Genre, Year, Duration
- **Source:** Compiled from personal Apple Music library export
- **License:** Public music metadata (song titles, artist names, album names, genres, years, durations). No copyrighted audio content is included.
- **PII (Personally Identifiable Information):** None. This dataset contains only public music metadata.

## sample_music.csv

- **Source:** Hand-curated sample subset for demonstration purposes.
- **License:** Same as above.
